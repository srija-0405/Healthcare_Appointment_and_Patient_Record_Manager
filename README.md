# 🏥 Healthcare Appointment & Patient Record Manager

A web-based **Healthcare Appointment and Patient Record Management System** developed as part of the **Software Engineering** course at **PES University**.

The application streamlines clinic operations by enabling patient registration, appointment scheduling, electronic medical record (EMR) management, billing, and role-based access for different users. The project was developed collaboratively following Agile software development practices.

---

## 📌 Project Overview

The Healthcare Appointment & Patient Record Manager is designed to simplify the daily operations of small healthcare clinics by providing an integrated platform for managing patients, appointments, medical records, billing, and administrative tasks.

The system supports multiple user roles with role-specific functionality to ensure secure and organized healthcare management.

## 🚀 Getting Started

### Prerequisites

- Python 3.10+
- pip

### Installation

Clone the repository

```bash
git clone https://github.com/srija-0405/Healthcare_Appointment_and_Patient_Record_Manager.git
```

Move into the project directory

```bash
cd Healthcare_Appointment_and_Patient_Record_Manager
```

Install the required packages

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
