## 📋 Project Description

A comprehensive healthcare management system for small clinics featuring patient registration, appointment scheduling, medical history tracking, and prescription management. This project emphasizes data security, CRUD operations, and professional software design patterns.

This repository contains the source code and documentation for the Healthcare Appointment and Patient Record Manager project, developed as part of the UE23CS341A course at PES University.

## 🛠️ Development Guidelines

### Branching Strategy
- `main`: Production-ready code
- `develop`: Development branch
- `feature/*`: Feature branches
- `bugfix/*`: Bug fix branches

### Commit Messages
Follow conventional commit format:
- `feat:` New features
- `fix:` Bug fixes
- `docs:` Documentation changes
- `style:` Code style changes
- `refactor:` Code refactoring
- `test:` Test-related changes

### Code Review Process
1. Create feature branch from `develop`
2. Make changes and commit
3. Create Pull Request to `develop`
4. Request review from team members
5. Merge after approval
