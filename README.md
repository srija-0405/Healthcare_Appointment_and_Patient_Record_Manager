# 🏥 Healthcare Appointment & Patient Record Manager

A web-based Healthcare Appointment and Patient Record Management System developed as part of the Software Engineering course at PES University.

The application streamlines clinic operations by enabling secure patient registration, appointment scheduling, electronic medical record management, and staff administration.

---

## ✨ Features

- Patient Registration
- Appointment Scheduling
- Patient Record Management
- Staff Management
- Secure Login Authentication
- CRUD Operations
- MySQL Database Integration

## 🚀 Getting Started

### Prerequisites

- PHP
- MySQL
- XAMPP

### Installation

```bash
git clone https://github.com/srija-0405/Healthcare_Appointment_and_Patient_Record_Manager.git
```

Start Apache and MySQL using XAMPP.

Import the SQL database into phpMyAdmin.

Open the project in your browser.
